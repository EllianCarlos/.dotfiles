# Kitty Configuration

This directory contains the configuration for the Kitty terminal emulator.

## Custom Keybindings

The `kitty_mod` is set to `CTRL+SHIFT`.

### Clipboard
- `CTRL+SHIFT+C`: Copy to clipboard
- `CTRL+SHIFT+V`: Paste from clipboard

### Window Management
- `CTRL+SHIFT+ENTER`: New window (split)
- `CTRL+SHIFT+W`: Close current window
- `CTRL+SHIFT+]`: Next window
- `CTRL+SHIFT+[`: Previous window
- `CTRL+SHIFT+F`: Move window forward
- `CTRL+SHIFT+B`: Move window backward
- `CTRL+SHIFT+L`: Next layout

### Tab Management
- `CTRL+SHIFT+T`: New tab
- `CTRL+SHIFT+Q`: Close current tab
- `CTRL+SHIFT+RIGHT`: Next tab
- `CTRL+SHIFT+LEFT`: Previous tab

### Font Size
- `CTRL+SHIFT+PLUS`: Increase font size
- `CTRL+SHIFT+MINUS`: Decrease font size
- `CTRL+SHIFT+BACKSPACE`: Reset font size to default

### Kittens (Plugins)
- `CTRL+SHIFT+P > F`: **Hints (Path)** - Select and copy/open file paths from the screen.
- `CTRL+SHIFT+P > U`: **Hints (URL)** - Select and open URLs in your browser.
- `CTRL+SHIFT+P > L`: **Hints (Line)** - Select and copy lines of text.
- `CTRL+SHIFT+P > W`: **Hints (Word)** - Select and copy individual words.
- `CTRL+SHIFT+U`: **Unicode Input** - Search and insert symbols or emojis.

## Appearance
- **Padding**: Window padding is set to `4` for a cleaner look.
- **Opacity**: Background opacity is set to `0.7`.
- **Font**: Using "Fira Code" with ligatures enabled.
